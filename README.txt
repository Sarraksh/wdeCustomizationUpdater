Для корректной работы необходимо внести в файл application.cfg следующие настройки:
CustomizationsFolder = "C:\WDE_custom_files"
WDEFolder = "C:\WorkspaceDesktopEdition\InteractionWorkspace"

где
CustomizationsFolder - папка, содержащая папки с кастомизациями
WDEFolder - папка установки WDE (та, что прописывается в момент установки приложения)

Если настройки указаны верно, wdeCustomizationUpdate.exe достаточно просто запустить (без каких-либо ключей), после чего

    Все файлы кастомизаций будут скопированы из своих папок в папку WDE
    При этом структура вложенных папок, таких как "Languages" будет сохранена
    В случае, если кастомизации содержат общие файлы, будет применён файл, который имеет самую позднюю дату изменения
    Создастся файл истории в директории программы, в котором будет список кастомизаций, их файлов и пользователь, под которым была запущена программа, а название этого файла будет содержать локальные дату и время запуска
    В реестр пользователя запишется ключ Software\Genesys\DeploymentManager\CustomFiles , содержащий данные о всех добавленных файлах кастомизаций
    По завершении программы будет запущен файл "InteractionWorkspaceDeploymentManager.exe" из папки WDE
    Если не требуется изменять какие-либо опции разворачивания ClickOnce можно будет просто нажимать "Далее" до завершения работы менеджера.
